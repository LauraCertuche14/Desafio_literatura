# Desafio_literatura
Proyecto desarrollado en Java utilizando Spring Boot para consultar libros desde una API y almacenarlos en una base de datos.

## Funcionalidades

- Buscar libros por título
- Guardar libros en base de datos
- Listar libros registrados
- Listar autores

## Tecnologías

- Java
- Spring Boot
- Maven
- PostgreSQL
